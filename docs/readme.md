# NextUp

NextUp is a platform connecting **Students** and **Organisations**.  
- Organisations can post **Opportunities** (internships, scholarships, events).  
- Students can **apply to opportunities**, **collaborate on projects**, and **exchange skills** using a coin‑based economy.  
- The system also supports **connections, transactions, notifications, and skillswaps** to build a social and learning network.

---

## 🚀 Tech Stack
- **Backend**: Node.js + Express  
- **Database**: MongoDB (Mongoose ODM)  
- **Auth**: JWT + OAuth (GitHub, LinkedIn)  
- **Deployment**: Vercel  
- **Testing**: Postman / BlazeMeter / Hoppscotch  
- **CI/CD**: GitHub Actions  

---

## ⚡ Features
- Student & Organisation accounts  
- Opportunities posting & applications  
- Collaborations between students  
- Skillswap with coin transfers  
- Transactions logging  
- Connections & notifications
